# Homebrew Tap for Ports Manager

This is the Homebrew tap for [ports-manager](https://github.com/adi-family/cli-ports-manager).

## Installation

```bash
brew tap adi-family/ports-manager
brew install ports-manager
```

Or install directly:

```bash
brew install adi-family/ports-manager/ports-manager
```

## Usage

After installation, run:

```bash
ports-manager --help
```

For more information, visit the [main repository](https://github.com/adi-family/cli-ports-manager).
